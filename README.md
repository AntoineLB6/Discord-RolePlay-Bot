# To launch the bot, you need to create a config.js and a serverSettings.json files, then, run `npm install` command for install all packages, then, to lanch the bot, run `node main.js`.

## Example config.js file :

```js
module.exports = {
    token: "<your discord bot token>",
    ownerID: "<your discord id>",
    DBCONNECTION: "<your mongodb adress>",
};
```
